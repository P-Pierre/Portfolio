Mes Competences

  • Communication

  • Data and application
  • Innovation and humanity
  • Innovative project

Smart Device

Introduction to Sensors

Compétences Niveau estimé/demandé Liens vers les projets/cours concernées

Microcontrollers and Open Source Hardware
